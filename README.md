# SIAM Kuis Helper

## Fitur

- Menampilkan popup kecil di kanan bawah halaman.
- Mengisi semua grup `radio` dengan nilai random `3`, `4`, atau `5`.
- Mengisi semua grup `radio` dengan nilai tetap `3`, `4`, atau `5`.
- Memindai jumlah grup radio dan menandai pertanyaan yang belum terisi.
- Tidak menekan tombol submit/kirim secara otomatis.
- Bisa dimatikan tanpa reload halaman.

## Cara Pakai

1. Buka halaman kuisioner,
2. Buka DevTools browser.
3. Masuk ke tab Console.
4. Paste script di bawah ini, lalu tekan Enter.
5. Gunakan popup `Local Kuis Helper` yang muncul di kanan bawah.
